# 贡献者行为准则 Code of Conduct

本项目采用 [Contributor Covenant](https://www.contributor-covenant.org)（贡献者公约）v2.1 作为社区行为准则，以下为要点摘录。

## 我们的承诺

为营造开放、友好的环境，我们承诺让每个人参与本项目与社区的体验都免于骚扰，无论其年龄、体型、身心障碍、族裔、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向。

## 我们的准则

有助于营造积极环境的行为包括：

- 使用友善、包容的语言；
- 尊重不同的观点与经历；
- 优雅地接受建设性批评；
- 以社区利益为重；
- 对其他成员保持同理心。

不可接受的行为包括：使用性化的语言或图像、人身攻击或侮辱性评论、公开或私下骚扰、未经许可发布他人隐私信息，以及其他有违职业操守的行为。

## 我们的责任

项目维护者有权也有责任删除、编辑或拒绝不符合本准则的评论、提交、代码、issue 及其他贡献，并会对其认为不当的行为说明理由。

## 适用范围

本准则适用于所有项目空间，以及个人代表项目或社区出现在公共场合时。

## 执行

如遇滥用、骚扰或其他不可接受的行为，可通过 GitHub 私信仓库维护者 [@Octo-o-o-o](https://github.com/Octo-o-o-o) 举报。所有投诉都会被审阅、调查，并作出被认为必要且恰当的回应；维护者有义务为举报者保密。

完整文本见 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。
